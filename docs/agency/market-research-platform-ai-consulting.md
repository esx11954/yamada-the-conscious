# プラットフォーム市場調査：AI活用コンサル / Claude Code実装 / AIエージェント講座

> 調査日：2026年4月12日
> 対象：ココナラ・ランサーズ・ストアカ

---

## Part 1 — ココナラ・ランサーズ：ChatGPT/AI活用コンサル

### 1-1. ボリューム・競合密度

| プラットフォーム | カテゴリ | 出品数・規模 |
|---|---|---|
| ココナラ | ChatGPT使い方・プロンプト作成 | **1,348件**（飽和気味） |
| ココナラ | AI導入・生成AI活用相談（上位カテゴリ） | 出品者2,500人以上、累計実績9,000件 |
| ランサーズ | AI・機械学習・ChatGPT（発注検索キーワード） | 2025年トレンド検索2位（需要あり、発注単価は分散） |

**AIカテゴリはピーク成長（前年比394%）を経て、汎用ChatGPT相談は飽和段階に入っている。**

### 1-2. 価格帯の実態（ココナラ）

| 内容 | 価格帯 | 競合密度 |
|---|---|---|
| プロンプト販売・教材 | 1,000〜3,000円 | 非常に高い |
| 1時間相談（汎用AI活用） | 3,000〜10,000円（**5,000円が最頻値**） | 高い |
| GPTs制作・カスタムAI | 10,000〜50,000円 | 中程度 |
| 業務自動化構築（n8n等） | 30,000〜180,000円 | **低い（穴場）** |

トップ出品者のレビュー数：129〜282件。ゼロからの差は大きく、初期は実績構築に全振りが必要。

### 1-3. ランサーズの構造変化（2025年発注トレンドランキング）

- AI・機械学習・ChatGPT：**2024年1位 → 2025年10位に急落**
- 意味：「ChatGPTを教える」需要がコモディティ化し、**実装・構築・監修への需要にシフト**した
- 2025年1位は「デザインデータ修正・変換」＝「AIが生成し、人が仕上げる」構造が定着
- ランサーズは2025年秋に **「Consul Partners for AI」**（AI案件特化の法人マッチング）を立ち上げ

### 1-4. 結論：初期営業として「1時間5,000円の汎用AI相談」をやる場合の注意点

| 判断軸 | 評価 |
|---|---|
| 競合環境 | 厳しい。1,300件超の競合あり、レビューゼロは選ばれにくい |
| 「顧客の悩みを集める場」としての有効性 | **有効。ただし業種絞り込みが前提** |
| 勝てるポジション | 「EC事業者向け」「製造業向け」等、業種×課題で絞ると競合が一桁になる |

## Part 2 — Claude Code実装：勝機の分析

### 2-1. 市場シグナル

**需要側：**
- Claude Code導入支援コンサルの需要が2026年に急増中
- AI導入支援は50,000〜100,000円の**新カテゴリ**として確立しつつある
- 「AIを使えるフリーランス」は月単価が約10万円高い傾向（ランサーズ調査）
- 企業の全社展開フェーズへの移行が始まっている（コミクス、Uravation等が導入支援に参入）

**供給側：**
- ランサーズ・ランサーズでのClaude Code実装代行出品数は**まだ少ない**
- ChatGPT相談と違い、「動くシステムを作る」スキルが必要なため参入障壁がある
- 競合はエンジニア寄りの出品者が多く、**ビジネス課題起点で話せる出品者がほぼいない**

### 2-2. 決定的なデータ：ストアカ Claude Code講座1,100人突破

株式会社Uravation（代表：佐藤傑氏、Xフォロワー10万人超）が開催した Claude Code実践講座：
- 形式：オンライン（Zoom）、3時間、プログラミング不要
- 価格：**4,000円**（定員300名×2日程）
- 申込者数：**1,100人突破**
- 内容：フォルダ整理の自動化、ドキュメント自動生成、ミニアプリ開発（初級）/ MCP連携、複数AI並列処理（発展）

→ 「プログラミング不要でClaude Codeを使いたいビジネスパーソン」の需要が証明されている。

### 2-3. Claude Code実装サービスの単価ポテンシャル

| サービス内容 | 想定単価 | 根拠 |
|---|---|---|
| Claude Code導入相談（1時間） | 15,000〜30,000円 | 汎用ChatGPT相談より専門性が高い |
| 社内ツール・自動化スクリプト構築 | 50,000〜150,000円 | AI導入支援の新カテゴリ相場 |
| 企業向け全社導入支援（月額） | 100,000〜300,000円 | エンジニア8名チームで導入支援事例あり |
| ストアカ講座（3時間） | 3,000〜8,000円 | 1,100人需要証明済み |

### 2-4. Claude Code × エージェンシー事業の勝機

**勝てる理由：**
1. **需要が急増しているのに供給が追いついていない**（ChatGPT相談と真逆の状況）
2. **「実装できる」＋「ビジネス課題を理解できる」の組み合わせが希少**
3. 初心者向け講座（ストアカ）から法人向け導入支援まで、**グレードの違うサービスラインを設計できる**
4. n8n/Make構築と組み合わせると「AIエージェントを作れる人」として完全差別化できる

**リスク：**
- Claude自体のアップデート速度が速く、スキルの陳腐化リスクあり（継続学習必須）
- 「プログラミング不要」という訴求が広まると参入者が増える可能性

## Part 3 — ストアカ：AIエージェント講座市場

### 3-1. プラットフォーム基本情報

- 掲載講座数：**8万件以上**（500ジャンル）
- 手数料構造：
  - 検索・ストアカ流入経由：**33%**（講師取り分67%）
  - リピーター・自己集客経由：**11%**（講師取り分89%）
- 価格設定：1,000円から講師が自由設定、入会金・別途料金の徴収不可

### 3-2. AI関連カテゴリの状況

| カテゴリ | レビュー平均 | 総レビュー数 |
|---|---|---|
| AI・機械学習 | 4.96 | 1,396件 |
| ChatGPT | 4.88 | 1,124件 |
| AI全般 | 4.89 | 457件 |

価格帯の実例：
- ChatGPT基礎：1,000円〜
- Claude/Copilot/Gemini活用：3,800円
- 動画ビジネス×AI：35,000円
- Claude Code実践（3時間）：**4,000円（申込1,100人）**

### 3-3. AIエージェント・n8n講座の状況

- ストアカ上の **AIエージェント特化講座はほぼ空白地帯**（2026年4月時点）
- n8nはConnpass・Udemy等では講座が増えているが、ストアカには未上陸に近い
- 「業務自動化 × AIエージェント」をビジネスパーソン向けに教える講座が存在しない

→ **最も競合が少なく、需要が確実にある穴場カテゴリ**

### 3-4. ストアカ参入戦略

| フェーズ | 内容 | 価格設定 |
|---|---|---|
| 入口（集客・実績構築） | 「Claude Codeで業務を自動化する入門講座（90分）」 | 1,000〜2,000円 |
| 本命（収益化） | 「EC事業者向けAIエージェント構築ハンズオン（3時間）」 | 5,000〜8,000円 |
| 高単価（継続収益） | 「マルチエージェント設計・n8n実装 全5回コース」 | 30,000〜50,000円 |

**手数料戦略**：SNS（X等）で自己集客できれば手数料11%。33%との差は大きく、自己集客ルートの整備が収益性の鍵。

## Part 5 — 推奨アクション

### 最初の90日で取るべきポジション

```
Week 1-4:  ストアカにClaude Code入門講座を出品（1,000〜2,000円・低リスク）
           → 受講者の「生の悩み」を収集する場として機能させる

Week 4-8:  ストアカ実績を元にECや製造業種向けの「業種特化AI相談」をココナラに出品
           → 汎用相談ではなく「EC事業者専門」等で絞り、競合を一桁にする

Week 8-12: ランサーズ「Consul Partners for AI」に登録
           → 法人案件を取りに行く（単価10万〜30万円のレンジ）
```

### 最も避けるべき出品

「ChatGPT活用コンサル（1時間5,000円）」を汎用的なタイトルで出品すること。
1,300件以上の競合に埋もれ、レビューゼロでは選ばれない。
